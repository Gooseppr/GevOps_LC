# Automatisation

Ce cours regroupe les savoirs et outils autour de l'automatisation des tâches système, réseau et infrastructure.

Il est indépendant de la formation La Capsule — les modules sont organisés par **thème** plutôt que par jour.

---

## Thèmes couverts

- **Orchestration** — Ansible, Terraform, configuration as code
- **Scripting** — Bash avancé, Python système, gestion des crons
- **Pipelines** — Intégration et déploiement continus hors CI/CD cloud
